# Лабораторная работа 1. Разработка пользовательского интерфейса (GUI) для языкового процессора  
## Цель работы:
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.  
## Студент: Осинцева Д.А. АП - 327  
## Описание проекта

Оконное приложение – текстовый редактор. Интерфейс содержит следующие области:
1) Основное меню программы.
2) Панель инструментов с кнопками для быстрого доступа к часто используемым функциям.
3) Область ввода/редактирования текста (текстовый редактор).
4) Область отображения результатов работы языкового процессора (только для вывода, без возможности редактирования).

**Меню "Файл":** Создание, открытие, сохранение, сохранение как, выход (с подтверждением сохранения изменений).

**Меню "Правка":** Стандартные операции с текстом: отмена/возврат, вырезание, копирование, вставка, удаление, выделение всего текста.

**Меню "Справка":** Вызов справочной системы и окна "О программе".

Все функции из меню доступны через соответствующие кнопки на панели инструментов.

C#

## Руководство пользователя

При запуске приложения открывается главное окно с текстовым редактором и областью отображения результатов работы языкового процессора.

При нажатии на кнопку "Создать" (Вкладка "Файл" пункт "Создать") откроется окно для выбора расположения, формата и названия нового файла.

Созданный файл откроется в редакторе текста.   
Открытие: Нажмите 'Открыть' для выбора файла (поддерживаются форматы: .txt, .doc, .csv)  
Сохранение: Нажмите 'Сохранить' для сохранения текущего файла.  
Сохранить как: Позволяет выбрать расположение, формат и название файла.  
Выход: Закрывает окно приложения. При выходе программа предложит сохранить изменения, если текст редактировался.

  <img width="400" height="400" alt="Подтверждение выхода" src="https://github.com/user-attachments/assets/7f73a115-5b3e-4894-90dd-df89f6a46625" />

"Копировать": Выделенный текст (или весь текст, если фрагмент текста не выделен) копируется в буфер обмена  
"Вырезать": Выделенный текст (или весь текст, если фрагмент текста не выделен) вырезается в буфер обмена  
"Удалить" Выделенный текст (или весь текст, если фрагмент текста не выделен) удаляется  
"Вставить": Текст из буфера обмена вставляется в окно редактора  
"Выделить все": Выделяется весь текст в окне редактора  
"Отменить": Отменяется последнее действие  
"Повторить": Возвращается отмененное действие
