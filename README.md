Лабораторная работа 1. Разработка пользовательского интерфейса (GUI) для языкового процессора
Цель работы:
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.
Студент: Осинцева Д.А. АП - 327
Описание проекта: 
Оконное приложение – текстовый редактор. Интерфейс содержит следующие области:
1) Основное меню программы.
2) Панель инструментов с кнопками для быстрого доступа к часто используемым функциям.
3) Область ввода/редактирования текста (текстовый редактор).
4) Область отображения результатов работы языкового процессора (только для вывода, без возможности редактирования).
Функционал текстового редактора включает:
Меню "Файл": Создание, открытие, сохранение, сохранение как, выход (с подтверждением сохранения изменений).
Меню "Правка": Стандартные операции с текстом: отмена/возврат, вырезание, копирование, вставка, удаление, выделение всего текста.
Меню "Справка": Вызов справочной системы (на данном этапе – описание реализованных функций) и окна "О программе".
Все функции из меню доступны через соответствующие кнопки на панели инструментов.
Используемые технологии: Язык программирования С#,Windows Forms,Visual Studio
Путь к готовому исполняемому файлу: "Lab1\bin\Debug\net8.0-windows\Lab1.exe"
Описание интерфейса и функций (руководство пользователя):
При запуско приложения открывается главное окно с текстовым редактором и областью отображения результатов работы языкового процессора
<img width="882" height="547" alt="image" src="https://github.com/user-attachments/assets/2f385af2-8ed3-43f9-9edc-80a128df1766" />
При нажатии на кнопку "Создать" (Вкладка "Файл" пункт "Создать") откроется окно для выбора расположения,формата и названия нового файла.
<img width="883" height="542" alt="image" src="https://github.com/user-attachments/assets/25e98a6f-2984-440c-887c-4ef62a7c2218" />
Созданный файл откроется в редакторе текста. При нажатии на кнопку "Сохранить" (Вкладка "Файл" пункт "Сохранить") текущий файл будет сохранен, при нажатии на пункт "Сохранить как" будет доступен выбор
расположения,формата и названия сохраняемого файла.
При нажатии кнопки "Открыть" откроется окно для выбора файла для открытия (формата .txt, .doc, .csv).
При нажатии на кнопку "Выход" приложение предлагает сохранить изменения в файле, если пользователь редактировал текст
<img width="877" height="542" alt="image" src="https://github.com/user-attachments/assets/fedf135f-9069-4dad-afcf-bbc6a49dc20d" />
При нажатии на кнопки "Копировать", "Вырезать","Удалить","Вставить" можно совершать операции с текстом (с выделенным фрагментом или со всем текстом, если не выделен определенный фрагмент).
Также реализована возможность выделить весь текст с помощью кнопки "Выделить все".
При нажатии на кнопки "Отменить" и "Повторить" реализован механизм отмены и повтора операций редактирования текста.
